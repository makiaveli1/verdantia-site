## Purpose

This pack supports Verdantia's flagship **Practical AI Workshop**: a half-day session for teams that need to turn scattered AI experimentation into safer, practical, repeatable working habits.

The pack is designed for mixed-confidence workplace teams using or evaluating tools such as Microsoft Copilot, ChatGPT, Claude, Gemini, Perplexity, or organisation-approved alternatives.

## Delivery rules

- Use synthetic, anonymised, or explicitly approved examples only.
- Do not ask participants to paste client, employee, personal, financial, legal, medical, confidential, or commercially sensitive data into public AI tools.
- If the organisation restricts live AI use, teach from prepared examples, screenshots, or offline demo material.
- The goal is better judgement and safer repeatable habits, not tool theatre.

## Workshop promise

By the end of the session, participants should understand:

1. Where AI tools can help.
2. Where they can mislead.
3. How to prompt with clearer context and constraints.
4. How to review AI output before relying on it.
5. Which team use cases are worth testing next.

## Standard session shape

1. Opening and baseline — 15 minutes
2. Tool literacy — 30 minutes
3. Prompting that survives real work — 40 minutes
4. Role-based practice — 55 minutes
5. Safe-use habits — 30 minutes
6. Use-case capture and next steps — 25 minutes

Core facilitation time: **195 minutes / 3h15**. Add a 10–15 minute break and light buffer to keep the full session inside the stated **3–3.5 hours plus breaks** window.

## Separate facilitator dependencies

This markdown pack does not yet include the offline deck, prepared demo examples, screenshot/video backups, or preloaded browser tab list. Those must be prepared separately before a live client session.
