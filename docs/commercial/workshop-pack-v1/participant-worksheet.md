## 3. Prompt structure that survives real work

Use this structure when asking AI for useful workplace output:

1. **Role** — what perspective should the AI take?
2. **Task** — what exactly should it do?
3. **Context** — what background does it need?
4. **Constraints** — what must it avoid or respect?
5. **Output format** — what should the answer look like?
6. **Review criteria** — how should the output be judged?

### Prompt template

```txt
Act as [role/context].

Task: [specific task].

Context: [audience, situation, goal, constraints].

Use only the information below. If something is missing, say what is missing instead of inventing it.

Constraints:
- Do not include confidential, personal, client, financial, legal, or sensitive information.
- Keep the tone [tone].
- Keep the output suitable for [audience].

Output format:
- [bullets/table/email/checklist/decision brief/etc.]

Before finalising, flag:
- assumptions
- risks
- anything I should verify
```

## 4. Bad prompt → better prompt

### Weak prompt

```txt
Write an email about the new process.
```

### Better prompt

```txt
Act as an operations manager writing to a mixed internal team.

Task: draft a clear email explaining a new expense approval process.

Context: the current problem is inconsistent submissions and missing receipts. The goal is to reduce back-and-forth and make the process easier to follow.

Constraints:
- Keep it calm and practical.
- Do not blame staff.
- Use plain English.
- Include a short bullet list of what changes from Monday.

Output format:
- subject line
- email body under 250 words
- 3-item checklist employees can follow

Before finalising, flag anything that should be confirmed with Finance.
```

## 5. Review before using AI output

Before using, sharing, or acting on AI output, ask:

- Is it accurate?
- Is it based on enough context?
- Did it invent facts, sources, numbers, or policies?
- Does it include private, sensitive, or confidential information?
- Is the tone right for the audience?
- Could this create bias, exclusion, or unfair treatment?
- Who is responsible for the final decision?
- What needs human review before this goes anywhere?
